# Projeto Prático – Bootcamp DIO

Este repositório apresenta um projeto prático desenvolvido como parte de um **bootcamp da DIO**. O objetivo foi utilizar o **NotebookLM** como ferramenta de **aprendizagem ativa**, a partir da curadoria de **3 a 5 fontes abertas** sobre um **tema financeiro introdutório**: **Renda Fixa para Iniciantes**.

O estudo foi conduzido por meio da definição de objetivos, elaboração de perguntas estratégicas e testes de diferentes variações de prompts, sempre com base no conteúdo das fontes selecionadas. Como resultado, foram produzidos **resumos estruturados**, um **glossário de conceitos financeiros** e um **conjunto de prompts reutilizáveis**, organizados neste repositório.

O foco do projeto está no desenvolvimento do **pensamento crítico**, na **organização do conhecimento** e no uso consciente da **Inteligência Artificial como apoio ao aprendizado**, e não como substituição do estudo.
